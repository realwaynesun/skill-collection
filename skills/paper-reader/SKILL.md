# 论文阅读 Skill

使用落合陽一教授的论文阅读框架，快速提取论文核心信息。

## 阅读流程

1. 获取论文内容（URL、PDF 或用户提供的文本）
2. 根据论文类型选择对应框架
3. 按框架逐项回答问题
4. 输出结构化总结

## 实验类论文框架

回答以下 6 个问题：

| # | 问题 | 要点 |
|---|------|------|
| 1 | 这是什么？ | 研究主题、问题定义 |
| 2 | 相比先前研究有何优势？ | 创新点、改进之处 |
| 3 | 技术/方法的核心是什么？ | 关键方法论、算法、模型 |
| 4 | 如何验证有效性？ | 实验设计、数据集、评估指标 |
| 5 | 有哪些讨论空间？ | 局限性、未解决问题、争议点 |
| 6 | 接下来读哪篇？ | 重要引用、延伸阅读建议 |

## 理论类论文框架

回答以下 4 个问题：

| # | 问题 | 要点 |
|---|------|------|
| 1 | 批评/讨论的对象理论是什么？ | 目标理论、学派 |
| 2 | 逻辑推理脉络是什么？ | 论证结构、关键论点 |
| 3 | 覆盖范围和内在一致性如何？ | 适用边界、自洽性 |
| 4 | 有哪些潜在分歧点？ | 可能的反驳、争议 |

## 输出格式

```markdown
# 论文总结：[论文标题]

**类型**：实验类 / 理论类
**领域**：[研究领域]
**发表**：[期刊/会议, 年份]（如有）

## 核心问题回答

[按框架逐项回答]

## 一句话总结

[用一句话概括论文贡献]

## 关键词

[3-5 个关键词]
```

## 注意事项

- 默认使用中文输出，除非用户指定其他语言
- 对于较长论文，优先关注 Abstract、Introduction、Method、Conclusion 部分
- 如论文类型不明确，默认使用实验类框架
